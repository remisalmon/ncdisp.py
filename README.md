# ncdisp.py
A Python implementation of MATLAB's [ncdisp](https://www.mathworks.com/help/matlab/ref/ncdisp.html) function.

## Usage

``` python
>>> from ncdisp import ncdisp
>>> ncdisp('example.nc')
```

## Requirements

`netcdf4>=1.5.3`

## Notes

The `location` and `dispFormat` keywords are not supported yet.
